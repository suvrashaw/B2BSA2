## Component Modification Requirements

For all components to work across pages, content must be passed via props instead of being hardcoded.

| Component | Required Props | Panels That Can Be Omitted |
|-----------|---------------|---------------------------|
| `HomeHero` | eyebrow, h1, description, ctas[], statsBar? | Stats bar row |
| `CinematicSequence` | frames[] (image/video src), storyBeats[] (text overlays) | All-or-nothing |
| `ClientLogos` | logos[] (src, alt, href) | ŌĆö |
| `WhoWeAre` | stats[] (value, label, description) | ŌĆö |
| `OurServices` | cards[] (title, description, image, href) | ŌĆö |
| `CaseStudies` | studies[] or categoryFilter | ŌĆö |
| `UpcomingEvents` | events[] or categoryFilter | ŌĆö |
| `WhyChooseUs` | reasons[] (title, body, image), sectionHeading | Right image panel |
| `Testimonials` | testimonials[] or categoryFilter | ŌĆö |
| `Blogs` | posts[] or categoryFilter | ŌĆö |
| `FAQ` | items[] (question, answer), sectionHeading | ŌĆö |
| `ContactUs` | heading, description, fields[]?, prefilledService? | Right illustration panel |
